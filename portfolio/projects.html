Fire Department Digital System

For a Capstone Project, we developed a system to help the London, AR Volunteer Fire Department transition from paper-based records to a digital solution. This project focused on building foundational software development and problem-solving skills. For the frontend we used Angular for rapid prototyping and the languages used were HTML, CSS, Typescript since the project was a web application. For the backened we used .NET for APIs and databases and MySQL for the storage.


  • Implemented call logging functionality - records emergeny call details such as incident tracking, location, and units dispatched.
  • Digitized fire dues tracking - Manages payments and records by providing a simple portal for residents and businesses
  • Improved record accessibility and organization
  • Reduced manual paperwork
